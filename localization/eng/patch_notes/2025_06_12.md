It's time for the second June patch! This week brings in more controller support so you should be able to play an entire run from start to finish using a controller or Steam Deck. Still, it's quite early so feedback on its ux, ui, bugs, and general feedback about its implementation are appreciated. The support is in for Xbox controllers so other controllers may experience some weirdness or incompatibility at this time. Please report those as issues too!

The Timeline is still taking a little break while we bring in some balance changes, ux fixes, wording updates, and the usual bug extermination.
