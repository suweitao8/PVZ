<@&1070463562008510616> Another Friday, another Byrdlett build! This one comes with some big Vlad--or should we say Zlatir--changes among various bug fixes, UI & UX additions, and balance changes!

**Heads up that we made some changes to our save file format which require your run history to be reset!** If you have an in-progress run, the game will auto-abandon it. So if you have an in-progress run you want to finish, you should finish it on your current build before downloading the new one.

Here are the steps to deleting your run history:

1. Open the command line with ~ key
2. Type "open saves"
3. Delete the settings.save, the progress.save, and the run_history folder
