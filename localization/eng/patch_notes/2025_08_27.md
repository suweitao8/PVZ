A new patch is here--a whole day early!! We're off to PAX West this week, so this is a slightly smaller one. There's no new content to speak of, but a lot of placeholder assets, writing updates, and UX/UI changes are here.

[blue][b]BALANCE:[/b][/blue]

Necrobinder:
- Buffed Eidolon card: no longer Exhausts, now has Retain
- Buffed Snap card: upgrade damage increased from +1 -> +3

Enemies:
- Buffed Exoskeletons: Hard to Kill power's HP loss per-turn cap decreased from 15 -> 9
- Changed Scroll of Biting: moveset is less randomized
- Nerfed Vine Shambler: Chomp damage decreased from 17(18) -> 16(17)

Events:
- Changed Brain Leech event: now uses card reward Rarity odds when you pick the Share Knowledge option

[blue][b]BUG FIXES:[/b][/blue]

General:
- Bound affliction no longer prevents cards from being auto-played
- Players can no longer click on rewards that are outside of the reward UI mask
- Effects that set a card's energy cost to a new absolute value (ex. Mummified Hand relic, Enlightenment card) now interact properly with effects that offset a card's existing cost (ex. Transfigure card)
- Fixed being able to keep the potion popup open when you open the pause menu via controller shortcuts
- Fixed card alternative hotkey button for non-combat "Choose a Card" screens
- Fixed controller input for "Give Up" confirmation screen
- Fixed issue with potion hovertip when you use a targeted potion via controller

Ironclad:
- Stampede card's autoplay effect now properly prevents Ripple Basin relic from triggering

Silent:
- Dash card now properly gains block before attacking, protecting the player if the enemy has Thorns power
- Fixed Murder base damage not being affected by the Corruption enchantment
- Murder card now only deals 7 damage turn 1

Defect:
- Cracked Core relic now properly triggers before Imbued enchantment

Necrobinder:
- Fixed Symbiosis card upgrade
- Reaper Form card no longer does stupid amounts of damage to invincible enemies

Regent:
- Fixed Sovereign Blade animation not working
- Sword Sage card no longer causes an infinite damage loop when it's triggered while you have Fencing Manual relic

Potions & Relics:
- Kifuda relic now lets you pick less than 3 cards to enchant
- Light Bulb relic no longer triggers when receiving card rewards outside of combat
- Unceasing Top relic is no longer triggered by card plays from effects that occur before the player draws their hand (ex. Mayhem card)

Ancients:
- Neow's Bundle choice now only appears if there are enough bundles with all of their cards unlocked

Mutiplayer:
- You can no longer access the multiplayer Daily/Custom unless you have unlocked the corresponding Epochs
- Merchant Robes relic no longer triggers for everyone in multiplayer
- Fur Coat relic no longer causes state divergences
- New version of Hard to Kill power no longer scales with player count
- Ping button now always animates out at end of combat
- Rewards icon no longer shows next to remote players after they're done taking rewards
- Reaction emotes no longer can block input
