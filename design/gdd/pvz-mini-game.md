> **Status**: Draft

> **Dependencies**: Main Menu System, Combat System (partial reuse)

---

## 1. Overview

植物大战僵尸迷你游戏是一个塔防小游戏，玩家在网格上放置植物来抵御僵尸的进攻。作为 Slay the Spire 2 的彩蛋小游戏，它提供了与主线玩法不同的轻松体验。

---

## 2. Player Fantasy

玩家想要一个轻松有趣的塔防体验，在卡牌构建的紧张节奏之外，享受经典 PvZ 风格的策略乐趣。植物和僵尸的设计融入 STS2 的世界观。

---

## 3. Detailed Rules

### 3.1 游戏场地

- **网格布局**: 5 行 x 9 列的经典草坪布局
- **阳光系统**: 玩家收集阳光作为资源
- **波次系统**: 僵尸分波次进攻，每波难度递增

### 3.2 植物系统

| 植物 | 阳光消耗 | 效果 | 冷却 |
|------|----------|------|------|
| 向日葵 | 50 | 产出阳光 | 快 |
| 豌豆射手 | 100 | 射击豌豆 | 中 |
| 坚果墙 | 50 | 阻挡僵尸 | 快 |
| 寒冰射手 | 175 | 射击冰豌豆，减速 | 中 |
| 樱桃炸弹 | 150 | 范围爆炸 | 慢 |
| 双发射手 | 200 | 双倍射速 | 中 |
| 大嘴花 | 150 | 一口吃掉僵尸 | 慢 |

### 3.3 僵尸系统

| 僵尸 | 生命值 | 速度 | 特殊能力 |
|------|--------|------|----------|
| 普通僵尸 | 100 | 慢 | 无 |
| 路障僵尸 | 200 | 慢 | 头戴路障 |
| 铁桶僵尸 | 400 | 慢 | 头戴铁桶 |
| 跑步僵尸 | 100 | 快 | 冲刺 |
| 巨人僵尸 | 1000 | 慢 | 投掷小僵尸 |

### 3.4 游戏流程

1. **准备阶段**: 玩家放置初始植物
2. **战斗阶段**: 僵尸从右侧进入，植物自动攻击
3. **波次递进**: 每波结束后短暂准备时间
4. **胜利条件**: 击败所有波次的僵尸
5. **失败条件**: 任何僵尸到达最左侧

### 3.5 阳光机制

- **初始阳光**: 50
- **天空掉落**: 每 5-10 秒掉落一个阳光 (25)
- **向日葵产出**: 每 24 秒产出 50 阳光
- **点击收集**: 玩家点击阳光收集

## 4. Formulas

### 伤害计算

```
植物伤害 = 基础伤害 * 等级加成
僵尸实际伤害 = 植物伤害 - 护甲 (最小为1)
```

### 阳光产出

```
天空阳光间隔 = 基础间隔 + 随机偏移(0-5秒)
向日葵产出间隔 = 24秒
```

### 波次难度

```
每波僵尸数量 = 波次编号 * 2 + 基础数量
僵尸类型概率 = 波次编号 / 总波次数
```

## 5. Edge Cases

| 情况 | 处理方式 |
|------|----------|
| 格子已有植物 | 显示禁止放置提示 |
| 阳光不足 | 显示阳光不足提示 |
| 所有植物被摧毁 | 游戏继续，允许重新种植 |
| 僵尸到达左侧 | 游戏失败，显示结算画面 |
| 暂停游戏 | 冻结所有行动，显示暂停菜单 |
